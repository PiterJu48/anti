# 동물복지인증 평가 시스템 구축 결과

농림축산식품부의 신뢰감 있는 이미지를 반영한 **동물복지인증 평가 시스템** 초기 버전 구축을 완료하였습니다.

## 주요 구현 사항

### 1. 디자인 시스템 (`style.css`)
- **공공기관 스타일**: Deep Blue(#003366)를 메인 컬러로 사용하여 정갈하고 신뢰감 있는 UI를 구축했습니다.
- **가독성**: Noto Sans KR 폰트와 넓은 여백을 활용하여 공무원 및 중장년층 사용자도 편리하게 사용할 수 있도록 설계했습니다.

### 2. 로그인 및 인증 로직 (`auth.js`, `index.html`)
- **단순 ID 로그인**: 사용자는 이메일 형식이 아닌 단순 ID로 로그인하며, 내부적으로 Supabase Auth와 호환되도록 변환 처리합니다.
- **Supabase 연동**: 제공해주신 URL과 Anon Key를 사용하여 실제 데이터베이스와 연동 가능하도록 설정했습니다.

### 3. 역할별 대시보드
- **관리자 (`admin.html`)**: 전체 통계, 최근 심사 현황, 사용자 관리, 평가항목 관리 기능을 포함합니다.
- **농장주 (`farm.html`)**: 현재 인증 상태 확인, 자가 진단, 신청 이력 조회 기능을 제공합니다.
- **평가원 (`evaluator.html`)**: 배정된 평가 목록 확인 및 보고서 작성 기능을 구현했습니다.

## 테스트 방법

1. `index.html`을 브라우저에서 엽니다.
2. 등록된 계정(ID/PW)으로 로그인을 시도합니다.
3. 로그인 성공 시 해당 역할의 페이지로 자동 이동하는지 확인합니다.

> [!NOTE]
> 실제 Supabase Auth에 사용자(ID@animal-welfare.mafra.go.kr 형식의 이메일)와 `user_metadata`에 `role`('admin', 'farm', 'evaluator') 정보가 등록되어 있어야 정상적으로 동작합니다.
